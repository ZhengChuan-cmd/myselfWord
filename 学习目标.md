  	第一次本地连接网站上的项目，需要先申请密钥，使用下面的命令
  	ssh-keygen -t ed25519 -C "your_github@example.com"
  	会在C:\Users\用户名称\.ssh的文件夹下面生成对应的密钥
  	把.pub结尾的文档内容复制到 github官网的setting中 SSH AND GPG KEYS 中new SSH key 中添加 
  	后就可以正常git clone
  	
  	设置你提交时的邮箱名称和用户名称
  	1.git config --global user.email "邮箱地址"
  	2.git config --global user.name "用户名称"
  	
  	创建一个分支
  	1.git branch 新分支名称
  	2.git checkout 新分支名称
  	3.git push -u origin 新分支名称
  	
  	提交内容
  	1.add 文件
  	可以使用 git add . add所有修改的文件
  	2.git commit -m"提交注释"
  	3.git push origin 分支名称
